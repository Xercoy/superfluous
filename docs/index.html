#002: Makefile

Make is a build automation tool. It reads directives from a file called a Makefile. People familiar with building or installing Linux application and tools from source will be very familiar with this. In Unix and Unix-like operating systems, make is oftentimes built in. You can create your own Makefile and have fun with it without needing to install anything. In addition to building applications, it can perform certain operations related to preparing and manipulating the project.
